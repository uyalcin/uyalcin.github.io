Projets de jeux-vidéos mobiles

Dual Ball

Dual Ball est un jeu de stratégie mobile 2D développé avec un camarade de mon école. Le principe est de résoudre les différents énigmes tout en étant le plus rapide possible.

Unity 3D,C#

Flabieber

Flabieber est un jeu 2D mobile fun qui a été développé à l'occasion de la flappy jam, à la suite du succès de flappy bird.

Unity 3D, C#

Projets de jeux-vidéos PC/autres

Uneris

Uneris est un jeu de survie/gestion 3D sur PC. L'objectif du jeu est de posséder un maximum de territoires dans un monde peuplé de mutants, en alternant entre le mode création et le mode première personne.

Unity 3D, C#

The Input Waves

The Input Waves est un jeu à deux joueurs PC développé en 48h à l'occasion de la Global Game Jam 2017. Le but du jeu est de construire son château de sable plus rapidement que son adversaire.

Unity 3D, C#

Projet d'application mobile

Kifu Snap

J'ai pu lors d'un stage aider Rémi Coulom à améliorer KifuSnap, qui est une application mobile permettant de déterminer le gagnant d'une partie de jeu de GO simplement à partir d'une photo.

Technologies utilisées : Android Studio, JAVA
Pour le traitement d'images : C++, Tensorflow
